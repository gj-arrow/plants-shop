# Plant Shop - Магазин растений

Современный интернет-магазин по продаже комнатных растений на Next.js.

## 🚀 Возможности

### Для покупателей
- 📦 Каталог товаров с фильтрацией по категориям
- 🛒 Корзина с сохранением в localStorage
- 📝 Оформление заказа с валидацией форм
- ✅ Страница подтверждения заказа

### Для администраторов
- 🔐 Авторизация в админ-панели
- 📊 Управление товарами (CRUD)
- 📋 Просмотр и управление заказами
- 🏷️ Изменение статусов заказов

## 🛠️ Технологии

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **База данных**: SQLite (better-sqlite3)
- **Авторизация**: bcryptjs + сессии на cookie

## 📁 Структура проекта

```
plant-shop/
├── src/
│   ├── app/
│   │   ├── api/          # API роуты
│   │   ├── admin/        # Админ-панель
│   │   ├── cart/         # Страница корзины
│   │   ├── checkout/     # Оформление заказа
│   │   └── ...
│   ├── components/       # React компоненты
│   ├── contexts/         # React Context (корзина)
│   ├── lib/              # Утилиты (БД)
│   └── middleware.ts     # Middleware для защиты роутов
├── scripts/
│   └── init-db.ts        # Скрипт инициализации БД
└── plant-shop.db         # SQLite база данных
```

## 🚀 Запуск

### Установка зависимостей
```bash
npm install
```

### Инициализация базы данных
```bash
npm run init-db
```

### Запуск dev-сервера
```bash
npm run dev
```

Сервер запустится на http://localhost:3000

### Продакшн сборка
```bash
npm run build
npm start
```

## 📍 Маршруты

| Маршрут | Описание |
|---------|----------|
| `/` | Главная страница с каталогом |
| `/cart` | Корзина |
| `/checkout` | Оформление заказа |
| `/order-success` | Подтверждение заказа |
| `/admin/login` | Вход для администратора |
| `/admin/products` | Управление товарами |
| `/admin/orders` | Просмотр заказов |

## 📝 API Endpoints

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/products` | Получить все товары |
| POST | `/api/products` | Создать товар (admin) |
| GET | `/api/products/[id]` | Получить товар по ID |
| PUT | `/api/products/[id]` | Обновить товар (admin) |
| DELETE | `/api/products/[id]` | Удалить товар (admin) |
| GET | `/api/orders` | Получить все заказы (admin) |
| POST | `/api/orders` | Создать заказ |
| PUT | `/api/orders/[id]` | Обновить статус заказа (admin) |
| POST | `/api/auth` | Вход/выход администратора |
| GET | `/api/auth` | Проверка сессии |
